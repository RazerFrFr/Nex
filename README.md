# Nex

Nex is a lightweight Fortnite cli launcher made in rust to be minimal and efficient.

## Features

- **Customization:** Nex have bundled tools like redirects, backends and gameservers customizable to the user preference and the option to set custom tools through settings.
  
- **Lightweight & Efficient:** the launcher itself is designed to be small, memory efficient and lightweight on the cpu.

- **Support for multiple versions:** Nex supports up to latest with proper tools.

## How to build from source

To build the launcher from source clone the repo and run

```bash
cargo build --release
```

Then get a zip from the current **Releases** and replace the old **nex.exe** with the new one from ``target/release``.
